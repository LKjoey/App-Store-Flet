## 使用方法

### 1. 安装 flet
```commandline
    pip install "flet[all]"
```

### 2. 运行
```commandline
    flet run main.py
```
或者 

直接在 pycharm 中运行 main.py 文件

### 其他

鼠标在  
top_bar() # 顶部栏 菜单栏和小铃铛之间的区域  中左键按下,
可以拖动窗口

按住 shift + 鼠标滚轮可以 左右滑动 browse_games 和 featured_games 区域中的内容
